# 🧠 Zyren — Origen y Significado del Nombre

## 🎯 Introducción

Zyren nace como la identidad cognitiva de la inteligencia artificial integrada en **Threshold**, una plataforma enfocada en el análisis, interpretación y transformación de material académico y didáctico en conocimiento estructurado y utilizable.

El nombre no surge como una palabra existente, sino como un **neologismo técnico y conceptual**, diseñado para transmitir precisión, inteligencia estructurada y transformación del conocimiento.

## 🚀 Capacidades de Zyren

Zyren no es un simple chatbot de pregunta-respuesta.

Es un **sistema inteligente multidimensional** capaz de transformar múltiples tipos de contenido académico en material de aprendizaje estructurado.

### 1. 💬 Chat Académico Interactivo

**Capacidad principal:**
Conversar en tiempo real sobre temas académicos con contexto persistente.

**Características:**
- 💬 Chat fluido y responsivo (velocidad optimizada con Groq)
- 🧠 Comprensión de contexto académico
- 📚 Mantiene historial de conversación
- 🎯 Respuestas orientadas al aprendizaje, no entretenimiento
- 🛡️ Protección contra prompts maliciosos (Shield Prompt)

**Ejemplo:**
```
Usuario: "¿Cómo funciona la fotosíntesis?"
Zyren: [Explica el proceso de forma estructurada y pedagogica]

Usuario: "¿Y la fase de oscuridad?"
Zyren: [Continúa con contexto de la pregunta anterior]
```

### 2. 🎴 Generación de [[FLASHCARDS_COMPLETE_DOCUMENTATION|Flashcards]] Automática

**Capacidad transformativa más potente de Zyren:**
Toma cualquier contenido y lo convierte en mazos de tarjetas de estudio.

**Características:**
- 🎴 Generación automática desde texto simple
- 📄 Extracción de flashcards desde documentos PDF grandes (hasta 100 MB)
- 🎥 Generación desde transcripciones de video
- 🔊 Generación desde grabaciones de audio
- 📊 Múltiples tipos: opción múltiple, verdadero/falso, rellenables
- 🎨 Soporte para código con resaltado de sintaxis
- 🔄 Detección automática cuando el usuario solicita crear mazos

**Tipos de flashcards generadas:**
```json
{
  "multiple_choice": {
    "pregunta": "¿Cuál es...?",
    "opciones": ["A", "B", "C", "D"],
    "respuesta_correcta": 0
  },
  "boolean": {
    "pregunta": "¿Es verdadero que...?",
    "respuesta": true
  },
  "fillable": {
    "pregunta": "El proceso se llama _____",
    "respuesta": "fotosíntesis"
  },
  "code": {
    "pregunta": "¿Qué hace este código?",
    "codigo": "```python\nprint('Hola')\n```",
    "respuesta": "Imprime 'Hola'"
  }
}
```

**Ejemplo en acción:**
```
Usuario: "Crea 10 flashcards sobre la célula"
Zyren: [Interpreta solicitud] → [Genera JSON automático] → [Persiste en BD] → [Retorna mazo]
```

### 3. 📖 Procesamiento de Documentos de Cualquier Tamaño

**Capacidad de análisis profundo:**
Zyren puede leer y procesar documentos académicos complejos.

**Características:**
- 📖 PDFs hasta 100 MB (usando Gemini Files [[API_DOCUMENTATION|API]])
- 📊 Extracción de información estructurada
- 🔍 Análisis profundo de contenido
- 📝 Resumen automático
- 🎯 Identificación de conceptos clave
- 💾 Cache temporal de documentos

**Casos de uso:**
- Subir apuntes en PDF → Zyren los analiza → Genera flashcards automáticas
- Cargar libro universitario → Zyren extrae puntos clave
- Procesar múltiples documentos → Síntesis integrada

**Límites y estrategia:**
```
Tamaño del documento:
├─ ≤ 50 KB  → Groq (ultrarrápido)
├─ 50 KB - 100 MB → Gemini (Files API)
└─ Respuesta: flashcards + resumen
```

### 4. 🎤 Transcripción de Audio (Grabaciones)

**Capacidad de conversión de voz:**
Convierte grabaciones de audio en texto estructurado.

**Características:**
- 🎤 Transcripción con Groq Whisper v3
- 🗣️ Soporte multiidioma (enfocado en español)
- ✍️ Formateo semántico automático
- 💾 Caching de transcripciones
- 🔄 Integración con generación de flashcards

**Flujo:**
```
Usuario graba clase/apuntes (m4a, mp3)
    ↓
Groq Whisper v3 transcribe
    ↓
Groq LLaMA 3 estructura semánticamente (opcional)
    ↓
Zyren genera flashcards automáticamente
```

**Ejemplo:**
```
Usuario graba explicación de profesor
Zyren: [Transcribe audio] → [Estructura párrafos] → [Detecta intención] → [Genera flashcards]
```

### 5. 📹 Transcripción de Videos YouTube

**Capacidad de integración multimedia:**
Extrae y procesa contenido educativo de YouTube.

**Características:**
- 📹 Extracción de subtítulos nativos
- 🗣️ Fallback a Whisper si no hay subtítulos
- 🌐 Soporte multiidioma
- 📚 Generación de resumen y flashcards
- 🔗 Referencias a timestamps

**Estrategia inteligente:**
```
Video de YouTube
    ↓
¿Tiene subtítulos en español?
    ├─ SÍ → Usar directamente
    └─ NO → Descargar audio + Whisper v3
    ↓
Procesar contenido
    ↓
Generar resumen y flashcards
```

### 6. 🖼️ Análisis de Imágenes y Gráficos

**Capacidad de visión por computadora:**
Interpreta imágenes académicas (gráficos, diagramas, pantallazos).

**Características:**
- 🖼️ Reconocimiento de gráficos y diagramas
- 📊 Extracción de datos de tablas
- 📐 Interpretación de ecuaciones y fórmulas
- 🔍 OCR de texto en imágenes
- 📸 Análisis de capturas de pantalla

**Casos de uso:**
- Subir gráfico de estadística → Zyren lo interpreta
- Foto de pizarrón → Zyren transcribe y estructura
- Captura de código → Zyren lo analiza

### 7. 📝 Generación de Resúmenes Inteligentes

**Capacidad de síntesis académica:**
Reduce contenido largo a puntos clave.

**Características:**
- 📝 Resumen de documentos completos
- 🎯 Extracción de conceptos clave
- 📊 Estructura de puntos principales
- 🔍 Preservación de contexto importante
- 🎨 Formateo legible

**Ejemplo:**
```
Documento de 50 páginas sobre biología
    ↓
Zyren genera resumen de 2-3 páginas
    ↓
Genera flashcards desde resumen
```

### 8. 🎯 Detección Inteligente de Intención

**Capacidad metacognitiva:**
Zyren entiende implícitamente lo que el usuario quiere hacer.

**Características:**
- 🎯 Detecta automáticamente solicitudes de mazos
- 🔄 Interpreta variaciones de lenguaje natural
- 🛡️ Valida intenciones académicas
- 📊 Genera acciones automáticas

**Ejemplo:**
```
Usuario: "Dame 15 flashcards sobre mitocondria"
Zyren: [Detecta intención] → %%DECK_ACTION%% [generación automática]

Usuario: "¿Podría hacer 20 tarjetas de biología?"
Zyren: [Interpreta solicitud] → [Genera mazos del contexto]
```

### 9. ⚡ Soporte Dual de Modelos (Groq y Gemini)

**Capacidad de optimización automática:**
Zyren selecciona inteligentemente entre dos modelos de IA.

**Características:**
- ⚡ Groq: 300+ tokens/segundo (respuestas rápidas)
- 📄 Gemini: 1M tokens contexto (documentos grandes)
- 🔄 Fallback automático si uno falla
- 💰 Optimización de costos
- 🎯 Selección contextual

**Lógica de decisión:**
```
¿Hay documento/PDF grande (>50KB)?
    ├─ SÍ → GEMINI (procesamiento profundo)
    └─ NO → GROQ (respuesta ultrarrápida)
```

### 10. 📂 Gestión de Mazos de Flashcards

**Capacidad organizativa:**
Zyren permite crear, organizar y recuperar mazos.

**Características:**
- 🏷️ Creación automática de mazos
- 📂 Organización por asignatura
- 🔍 Búsqueda y recuperación
- 📊 Estadísticas de estudio
- 🔄 Importación/exportación JSON

**Estructura:**
```json
{
  "id": 1,
  "title": "Fotosíntesis",
  "description": "Generado desde apuntes de biología",
  "subject_id": 2,
  "cards": [
    { "type": "multiple_choice", "data": {...} },
    { "type": "boolean", "data": {...} }
  ],
  "created_at": "2026-05-26",
  "card_count": 15
}
```

### 11. 🛡️ Protección y Validación (Shield Prompt)

**Capacidad de [[SECURITY|seguridad]]:**
Zyren valida y protege contra prompts maliciosos o fuera de contexto.

**Características:**
- 🛡️ Detección de jailbreak attempts
- ⚠️ Filtrado de contenido inapropiado
- ✅ Validación de intención académica
- 🔒 Limitación de contexto
- 📋 Restricción de respuestas fuera de scope
